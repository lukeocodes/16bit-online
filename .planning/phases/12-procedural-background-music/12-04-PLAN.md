       - `handleTransition(from: MusicState, to: MusicState): void`:
         1. Look up track: `const factory = this.trackRegistry.getTrack(to, this.currentZoneTag)`

         4. Stop and dispose any existing track on the inactive side: `if (activeTracks.has(side)) { activeTracks.get(side)!.dispose(); }`

         9. Store: `activeTracks.set(side, track)`
         10. If entering combat/boss, save current Transport BPM as baseBPM before combat track changes it

       - `getActiveTrackId(side: "a" | "b"): string | null`:
         Returns the id of the track on the given side, or null.

       - `getTrackRegistry(): TrackRegistry`:
         Returns the internal track registry (for dev API to call getAllTrackIds).

       - `dispose(): void`:
         Stop and dispose all active tracks. Call sampleCache.disposeAll(). Clear activeTracks map. Clear trackRegistry.
