       ```typescript
       function getSamplerUrls(): Record<string, string> {
         const urls: Record<string, string> = {};
         for (const oct of OCTAVE_RANGE) {
           for (const note of SPARSE_NOTES) {

         }
         return urls;
       }
       ```

    **PhraseEngine.ts** — Create at `packages/client/src/audio/music/PhraseEngine.ts`:

    1. Import `* as Tone from "tone"`.

    2. `PhraseEngine` class with:
       - `constructor(phrasePool: string[][], subdivision: string = "8n")` — stores pool and subdivision
       - Private `currentPhraseIndex: number` — initialized to random index
       - Private `lastPhraseIndex: number` — tracks previous to avoid immediate repeat
       - Private `sequence: Tone.Sequence | null`
       - `selectNextPhrase(): string[]` — picks random index from pool, re-rolls once if same as lastPhraseIndex (when pool.length >= 2). Updates lastPhraseIndex. Returns the selected phrase.
       - `createSequence(sampler: Tone.Sampler, velocity: number = 0.7): Tone.Sequence` — creates a Tone.Sequence that iterates through step indices, triggers sampler.triggerAttackRelease for each non-null note in the current phrase, and calls selectNextPhrase() at the last step. Sets sequence.loop = true. Stores as this.sequence.
       - `getCurrentPhrase(): string[]` — returns the current phrase
       - `dispose(): void` — stops and disposes the sequence if it exists, sets to null

    3. The Tone.Sequence callback:
       ```typescript
       new Tone.Sequence(
         (time, stepIndex) => {
           const phrase = this.phrasePool[this.currentPhraseIndex];
           if (stepIndex < phrase.length && phrase[stepIndex] !== null) {
             sampler.triggerAttackRelease(phrase[stepIndex], this.subdivision, time, velocity);
           }
           if (stepIndex === phrase.length - 1) {
             this.selectNextPhrase();
           }
         },
         Array.from({ length: this.phrasePool[0].length }, (_, i) => i),
         this.subdivision
       );
       ```
