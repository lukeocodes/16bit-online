       interface ProximityStem {
         name: string;
         gainNode: GainNode;
         triggerDistance: number;  // starts fading in at this Manhattan distance
         fullDistance: number;     // full volume at this distance (closer)
       }

    2. `ProximityMixer` class with:
       - Private `stems: ProximityStem[]` array
       - `addStem(name: string, gainNode: GainNode, config: { triggerDistance: number; fullDistance: number }): void` — adds stem to array, sets initial gain to 0
       - `update(playerPos: { x: number; z: number }, poi: { x: number; z: number }): void` — for each stem, calculate Manhattan distance `abs(playerPos.x - poi.x) + abs(playerPos.z - poi.z)`, then compute gain as:
         ```
         t = clamp(1 - (dist - fullDistance) / (triggerDistance - fullDistance), 0, 1)
         ```
         Use AudioParam automation: `gainNode.gain.setValueAtTime(current, now); gainNode.gain.linearRampToValueAtTime(t, now + 0.3);`
       - `dispose(): void` — sets all stem gains to 0, clears array
